# t265_json


## Clone & build using catkin

```
mkdir ~/catkin_ws/src -p
cd ~/catkin_ws/src
git clone  https://github.com/chrissunny94/t265_json
cd ../../
catkin build
source devel/setup.bash
```


## How to use


```
roslaunch t265_json t265_node_without_rviz.launch

```
